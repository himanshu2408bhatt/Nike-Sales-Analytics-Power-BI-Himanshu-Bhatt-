# Nike-Sales-Analytics-Power-BI-Himanshu-Bhatt-
##  Project Summary
Built an interactive **Power BI dashboard** using the Nike Sales dataset from Kaggle.  
The project demonstrates end-to-end BI skills: **data cleaning, star schema modeling, DAX measures, and dashboard design**.

##  Data Model
- **Fact Table:** Nike_Sales_Fact (Revenue, Profit, Units Sold, Discounts, etc.)
- **Dimensions:** Date, Product, Region
- **Relationships:** 3 active (Fact â†’ Date, Product, Region)

##  Key Measures
- 40+ DAX measures including:
  - **Revenue & Profit KPIs**
  - **Profit Margin %**
  - **Units Sold**
  - **Online vs Retail performance**
  - **Time Intelligence (YTD, QTD, YoY)**
  - **Profitability ratios & rankings**

##  Dashboard Highlights
- **Top KPIs:** Revenue, Profit, Margin %, Units Sold, Profitable Transactions
- **Channel Analysis:** Online vs Retail sales comparison
- **Regional Insights:** Profit and Units Sold by city (Delhi, Bangalore, Hyderabad, Pune, Kolkata)
- **Time Trends:** Revenue heartbeat chart by Year, Quarter, Month, Day
- **Profitability:** Pie chart showing profitable vs non-profitable transactions
- **Interactive Filters:** Region, Product, Sales Channel

## Business Value
- Quick executive snapshot of performance
- Compare **channels, products, and regions**
- Track **profitability and data quality**
- Identify **seasonality and growth trends**
